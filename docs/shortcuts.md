## Mouse Controls

### The Canvas
* **Left Click:** Place object (if a tool is active) or Select object.
* **Click & Drag (Background):** Create a selection box to select multiple items.
* **Right Click:** Open the **Context Menu** for the object under the cursor (Edit, Delete, Properties).
* **Scroll Wheel:** Pan the view up/down.

### Object Interaction
* **Shift + Click:** Add/Remove an object from the current selection.
* **Drag Selection:** Moves all selected objects together.

:::tip Pro Tip: Group Rotation
You can select multiple bales (by clicking and dragging a selection box or Shift+Clicking individual objects) and press `R` to rotate the entire group at once. This is perfect for creating angled tunnels or complex structures.
:::
